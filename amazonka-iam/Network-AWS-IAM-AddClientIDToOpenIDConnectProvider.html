Description

Adds a new client ID (also known as audience) to the list of client IDs already registered for the specified IAM OpenID Connect provider. This action is idempotent; it does not fail or return an error if you add an existing client ID to the provider.

http://docs.aws.amazon.com/IAM/latest/APIReference/API_AddClientIDToOpenIDConnectProvider.html
